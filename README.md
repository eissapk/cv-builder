# CV Builder

Create professional resumes, CV online for free, in minutes. Simply fill in your details and generate beautiful PDF and HTML resumes!

## Features
* Preview of Finished CV
* Slider
* Simple Form for filling out your data
* Pure Code
* Animations
* Elegant Design
* Fast Loading
* Cross-Browser
* Fully Responsive
* Much More
